* hay dos tipos de despliegue en GitHub Pages
  * Usuario /Organización Pages (https://<USERNAME|ORGANIZATION>.github.io/)
  * Proyecto Pages (https://<USERNAME|ORGANIZATION>.github.io/<PROJECT>/)
* para decidir cual de los dos métodos es mejor seguir este [enlace](https://help.github.com/en/articles/user-organization-and-project-pages#user--organization-pages)

# proyecto pages

* es el que menos compromete pues puede tener tantos como proyectos tengas
