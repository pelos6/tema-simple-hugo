* para poder usar emoticonos hay que poner en el archivo de configuración esta línea

```toml
# para poder usar emoticonos
enableEmoji = true
```

* se pueden poner los que salen en [esta lista](https://www.webpagefx.com/tools/emoji-cheat-sheet/)
